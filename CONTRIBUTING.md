# Contributing

This is a personal learning tool, not actively maintained as a community project. That said, issues and pull requests are welcome.

## Getting Started

See [README.md](README.md) for local setup and [CLAUDE.md](CLAUDE.md) for development conventions, project structure, and architectural decisions.

## Before Opening a PR

- Keep changes focused and scoped (fix the bug, add the feature — don't also refactor surrounding code)
- If you're changing behavior, explain why in the PR description
- The codebase uses Bun, not npm/yarn — use `bun install`, `bun run`, etc.

## Issues

Bug reports and feature suggestions are welcome. No guarantees on response time or whether changes will be merged.
